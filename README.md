# Генератор документов AVA Solution

Веб-приложение для генерации договоров перевозки и транспортных заявок (.docx)
из шаблонов, с базой договоров, справочником контрагентов и интеграцией
с реестром перевозок (Google Форма).

Работает на **PHP** (виртуальный хостинг PS.kz / Plesk, без Node.js и БД).
Продакшн: <https://docs.ava-solution.kz> (документ-рут указывает на папку `php/`).

## Структура

```
php/
  index.html        — весь интерфейс (вкладки: Договор, Заявка, База договоров)
  .htaccess         — запрет доступа к data/ + маршрутизация /api/* (если rewrite включён)
  api/index.php     — фронт-контроллер API (роутинг через ?r=..., работает без rewrite)
  lib/
    companies.php   — реквизиты наших компаний
    helpers.php     — даты, номера, сумма прописью, заполнение DOCX
    docgen.php      — сборка договора/заявки (используется и при пересоздании)
    storage.php     — JSON-хранилища: документы, база договоров, контакты, счётчики
    auth.php        — вход по паролю + защита от перебора
  templates/        — шаблоны .docx с метками {ПОЛЕ}
  data/             — данные (не в git): docs.json, deals.json, contacts.json,
                      counters.json, auth.json, sessions/
```

## Ключевые решения

- **Документы не хранятся файлами**: сохраняются исходные данные (docs.json),
  .docx пересобирается при каждом скачивании — диск не расходуется.
- **Номера уникальны в рамках дня**: второй документ за день получает суффикс `/2`.
- **Роутинг через `?r=`**, потому что rewrite-правила на хостинге не применяются.
- **Часовой пояс** зафиксирован: Asia/Almaty.
- Резервная копия (кнопка в интерфейсе) включает всё: базу, контакты, документы, счётчики.

## Локальный запуск

```bash
cd php
DATA_DIR=/tmp/cg-data php -S 127.0.0.1:8080
# открыть http://127.0.0.1:8080 (первый вход — установка пароля)
```

## Деплой

Push в `main` → GitHub webhook → Plesk Git автоматически подтягивает изменения.
Подробности: `php/PLESK-DEPLOY.md`.
